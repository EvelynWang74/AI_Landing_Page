## JavaScript
Single IIFE in `assets/js/main.js` (`'use strict'`). Five behaviours:
1. Auto-year in footer.
2. Scroll reveal — immediate (no animation, carousel layout made IntersectionObserver unreliable).
3. Floating bar — code retained but bar removed from HTML.
4. Gallery carousel — thumb click → main image fade swap.
5. Slide nav dots — click to scroll panel into view; IntersectionObserver updates active dot.
6. Smooth scroll for anchor links.
7. Reduced-motion — zeroes all duration tokens if OS preference set.

**Bug fixed today:** A stray uncommented text block inside the IIFE caused a `SyntaxError` in strict mode, silently preventing the entire script from running. Fixed by removing the orphaned text (lines 138–140 in the original file).
