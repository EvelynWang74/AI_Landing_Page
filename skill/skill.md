# Collaboration Skills & Rules

This file defines how we work together on this project.
These rules apply to every step, every file, every decision.

---

## Rule 1 — Ask, Never Guess

> **If there is anything I don't know — content, intent, preference, or a detail that belongs to you — I will stop and ask. I will never invent, assume, or fill in blanks on your behalf.**

### What this covers
- Your name, handle, bio copy, headline wording
- Your actual social media URLs
- Which photo goes in which gallery slot
- Any preference not already stated in this project
- Design decisions where multiple valid options exist and your taste is the deciding factor

### What this does NOT block
- Standard web conventions (semantic HTML, CSS resets, accessibility patterns)
- Implementation details that have a clear best practice
- File/folder structure decisions already approved by you

---

## Rule 2 — Step-by-Step with Approval Gates

No step begins without your explicit "yes".
Each step is summarised before execution so you know exactly what is about to happen.

---

## Rule 3 — No Surprise Changes

If a decision made in a previous step needs to change, I flag it and explain why before touching anything.

---

## Rule 4 — Content Is Yours

All copy (headlines, bio, captions, alt text) starts as a clear placeholder.
Nothing goes live with made-up content. You fill in the words; I build the container.
